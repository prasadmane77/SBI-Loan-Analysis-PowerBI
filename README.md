# SBI-Loan-Analysis-PowerBI
Power BI dashboard project for analyzing loan performance ,fraud flags and  trends using  dummy SBI loan data

## Overview
Power BI dashboard built using simulated loan data to analyze loan portfolio performance, fraud indicators, and disbursement trends.

---

## Key KPIs
- Total Loans: 300  
- Total Loan Amount: 311.24 Million USD  
- Average Loan Amount: 1.04 Million USD  

---

## Key Insights
- Active loans dominate the portfolio (88.33%)
- Defaulted loans account for 7.33%
- Mid and Top categories contribute the highest loan volume
- 137 loans flagged as fraud suspected
- Highest loan disbursement observed in July
- Seasonal variation seen in monthly loan trends

---

## Tools Used
- Power BI  
- Power Query  
- DAX
